# Changelog

## [1.1.0] - 2026-05-13

### Added
- Hot reload: jitsu functions are now re-imported fresh on every pipeline run without needing a server restart (`moduleCache: false` on jiti instances)

## [1.0.0] - Initial release

- Local debugging playground for Jitsu transformation functions
- Browser UI with pipeline builder, input payload editor, env vars, and geo context
- Dynamic TypeScript function loading via jiti
- Step-by-step execution results with logs and duration
